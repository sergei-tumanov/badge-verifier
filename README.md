# Badge Verifier

Web-инструмент для верификации авто-нарезки аудиобейджей продавцов.

**Production:** https://badge-verifier.vercel.app

## Что это

Длинные аудиозаписи с бейджей продавцов автоматически режутся на отдельные клиентские диалоги. Этот UI помогает проверить:

- Не пропустил ли разбиватель какие-то клиентские диалоги
- Корректно ли определил границы (start/end) каждого клипа
- Сверить транскрипцию длинного файла vs нарезанных клипов side-by-side

## Возможности

- 📑 **Side-by-side просмотр** длинной транскрипции и нарезанных клипов с алайнментом по времени
- 🎵 Аудио-плеер длинного файла со скоростью и абс. таймкодами
- ✏️ Inline-правки текста, ролей, таймкодов сегментов
- 🏷 Ручная разметка диалогов (ground truth) с авто-метриками miss/extra/split/merge
- 🤖 Поддержка LLM-разметки через импорт JSON
- 💾 Авто-сохранение в localStorage по audio_id
- 📥📤 Импорт/экспорт правок и диалогов
- 📱 Мобильная версия

## Стек

- Pure HTML + Vanilla JS (нет билда, нет фреймворков)
- Деплой: Vercel static hosting
- Данные: JSON-файлы рядом с `index.html`

## Структура

```
├── index.html                       — само приложение
├── badges_metadata.json             — 62 коротких клипа (vt_smart_*.wav)
├── long_sessions_metadata.json      — 7 длинных сессий (vt_session_*.wav)
└── vercel.json                      — конфиг деплоя
```

## Управление

### Десктоп

- Селект **Менеджер** + **Сессия** → автозагрузка
- Одиночный клик по строке → контекстное меню (играть, роль, тайминги, split, merge, удалить, скорость)
- Двойной клик по тексту → редактирование
- `Enter` в тексте → split на месте курсора
- `Backspace` в начале → merge с предыдущим
- `J` / `K` → следующий miss / extra
- `Space` → play/pause
- `D` → разметить новый диалог

### Мобилка

- 📑 кнопка в шапке → drawer с диалогами/метриками/клипами
- Тап → контекстное меню
- Двойной тап → редактировать текст
- FAB `+` → разметить диалог
